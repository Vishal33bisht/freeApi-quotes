# 💬 Free API Quotes App

A simple and modern Quotes Generator web application built using HTML, CSS, and JavaScript.  
This project fetches random quotes from an API and displays them dynamically with a clean and responsive user interface.

---

## 🚀 Features

- ✨ Fetch random quotes dynamically
- 📜 Display quote text and author
- 🎨 Clean and responsive UI
- ⚡ Lightweight and fast application
- 🔄 Generate new quotes instantly
- 📱 Mobile-friendly design

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- REST API

## ⚙️ Installation & Setup

Clone the repository:

```bash
git clone https://github.com/Vishal33bisht/freeApi-quotes.git
```

Move into the project directory:

```bash
cd freeApi-quotes
```

Open the project in your browser:

```bash
open index.html
```

Or simply double-click `index.html`.
